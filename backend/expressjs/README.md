
# Express Project

이 프로젝트는 Node.js와 Express를 기반으로 하는 웹 애플리케이션입니다.

## 개발 환경 설정
- **Runtime Manager:** [mise](https://mise.jdx.dev/)
- **Package Manager:** [pnpm](https://pnpm.io/)

## 시작하기

1. **의존성 설치**
# Node.js LTS 버전 설치 및 활성화
mise use node@lts

# 설치 확인
node -v

# pnpm 설치 및 활성화
mise use pnpm

2. **서버 실행**
   \`\`\`bash
   node app.js
   \`\`\`

## 프로젝트 구조
- \`app.js\`: 메인 서버 파일
- \`package.json\`: 프로젝트 정보 및 의존성 관리
- \`.gitignore\`: Git 제외 항목 설정
